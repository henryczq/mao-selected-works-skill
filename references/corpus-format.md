# Corpus Format

推荐把《毛选》整理成"一篇文章一个 Markdown 文件"。

最理想格式：

```yaml
---
volume: 第一卷
article_title: 中国社会各阶级的分析
aliases:
  - 阶级分析
date: 1925-12-01
tags:
  - 阶级
  - 中国革命
---
```

正文里建议保留：

- 一级标题：文章标题
- 二级标题：分节标题
- 自然段

推荐目录：

```text
data/
├── 01-01-中国社会各阶级的分析.md
├── 01-02-湖南农民运动考察报告.md
└── ...
```

命名规范：`{卷号}-{序号}-{文章标题}.md`，如 `01-07-反对本本主义.md` 对应第一卷第七篇。
